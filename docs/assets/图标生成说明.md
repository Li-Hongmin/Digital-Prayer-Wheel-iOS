# 应用图标生成说明

## 方法一：使用 Xcode Preview（推荐）

### 步骤：

1. **在 Xcode 中打开** `AppIconGenerator.swift` 文件
   - 位置：`Digital-Prayer-Wheel-iOS/Utils/AppIconGenerator.swift`

2. **启动 Preview**
   - 点击右上角的 "Resume Preview" 按钮
   - 或按快捷键 `⌥⌘P`

3. **选择预览模式**
   - 在 Preview 中选择："标准图标 - 1024x1024"
   - 可以看到完整的图标设计

4. **截图保存图标**
   - macOS 截图快捷键：`⌘⇧4` 然后按空格
   - 点击 Preview 窗口进行截图
   - **重要**：确保截图大小为 1024x1024 像素

5. **导入到 Xcode**
   - 打开 `Assets.xcassets`
   - 点击 `AppIcon`
   - 将截图拖入 "iOS" 的 1024x1024 格子
   - 对深色模式和 Tinted 模式重复此操作

## 方法二：使用导出工具（需要运行应用）

### 步骤：

1. **临时添加导出入口**
   在 `iOSContentView.swift` 中添加：
   ```swift
   // 临时：在某个地方添加导出按钮
   NavigationLink("生成图标") {
       IconExportView()
   }
   ```

2. **运行应用**
   - 在模拟器或真机上运行
   - 点击"生成图标"按钮

3. **导出图标**
   - 点击"导出全部三个图标"
   - 图标会保存到相册
   - 从相册导出到电脑

4. **导入 Xcode**
   - 将图标拖入 `Assets.xcassets/AppIcon`

## 方法三：手动创建（最简单）

如果上述方法复杂，我可以帮你：

1. **使用在线工具**
   - 使用 Figma、Canva 等工具
   - 参考 `AppIconGenerator.swift` 中的设计

2. **图标设计元素**
   - 背景：深红到金色渐变
   - 外圈：金色双环装饰
   - 中心：白色卍字符（字体大小约 480pt）
   - 尺寸：1024x1024px
   - 圆角：iOS 自动处理

## 图标设计说明

### 标准模式图标
- 背景：红金渐变（体现庄严温暖）
- 主体：金色转经轮
- 中心：白色卍字符
- 装饰：多层金色圆环

### 深色模式图标
- 背景：深灰色（适配深色界面）
- 主体：金色发光转经轮
- 中心：金色卍字符（发光效果）
- 装饰：金色圆环

### Tinted 模式图标
- 背景：纯金色
- 主体：白色轮廓
- 中心：白色卍字符
- 简化设计

## 图标文件命名建议

生成后的文件建议命名为：
- `AppIcon.png` - 标准模式
- `AppIcon-Dark.png` - 深色模式
- `AppIcon-Tinted.png` - Tinted 模式

## 注意事项

1. **尺寸**：必须是 1024x1024 像素
2. **格式**：PNG 格式，无透明度
3. **圆角**：不要手动添加圆角，iOS 会自动处理
4. **内容**：避免边缘内容，保持安全边距（约 10%）

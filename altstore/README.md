# 至诚转经轮 - AltStore 源

藏传佛教修行助手的 AltStore 分发源。

## 📱 如何安装

### 第 1 步：安装 AltStore

1. 访问 [altstore.io](https://altstore.io) 下载 AltStore
2. 按照官方教程在 iOS 设备上安装

### 第 2 步：添加本源

1. 打开 AltStore 应用
2. 点击 `Sources` 标签
3. 点击右上角 `+` 号
4. 输入源地址：
   ```
   https://li-hongmin.github.io/Digital-Prayer-Wheel-iOS/altstore/apps.json
   ```
5. 点击 `Add`

### 第 3 步：安装应用

1. 进入 `Browse` 标签
2. 找到「至诚转经轮」
3. 点击 `INSTALL`

## ✨ 应用功能

- 🙏 **四种经文**：六字大明咒、心经、南无阿弥陀佛、南无观世音菩萨
- 📈 **指数增长计数系统**：支持极大数值记录
- 🎨 **自定义弹幕显示**：多种显示方式和自定义设置
- 💾 **自动保存进度**：持久化存储修行进度
